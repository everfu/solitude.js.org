## Introduction au Front Matter

Le Front Matter est une section de métadonnées YAML située au début de vos fichiers Markdown. Il permet de définir des propriétés spécifiques pour chaque article ou page.

## Structure de base

::tabs{variant="card"}
  ::div{label="Article de base" icon="vscode-icons:file-type-markdown"}

  ```yaml
  ---
  title: Titre de l'article
  date: 2024-01-15 10:30:00
  categories: [Développement]
  tags: [hexo, blog, tutorial]
  description: Description courte de l'article
  ---
  
  Contenu de l'article ici...
  ```
  ::
::

## Propriétés communes

### Propriétés obligatoires

::tabs{variant="card"}
  ::div{label="Propriétés de base" icon="lucide:list"}

  ```yaml
  ---
  title: "Titre de votre article"           # Titre affiché
  date: 2024-01-15 10:30:00                # Date de publication
  ---
  ```
  ::
::

### Propriétés optionnelles

::tabs{variant="card"}
  ::div{label="Propriétés étendues" icon="lucide:settings"}

  ```yaml
  ---
  title: "Mon article de blog"
  date: 2024-01-15 10:30:00
  updated: 2024-01-16 14:20:00            # Date de mise à jour
  categories: [Développement, Web]          # Catégories
  tags: [javascript, react, tutorial]      # Tags
  description: "Description pour SEO"      # Description meta
  keywords: [js, react, développement]     # Mots-clés SEO
  author: "Votre nom"                      # Auteur spécifique
  permalink: mon-article-personnalise      # URL personnalisée
  ---
  ```
  ::
::

## Propriétés spécifiques au thème Solitude

### Configuration de l'affichage

::tabs{variant="card"}
  ::div{label="Affichage" icon="lucide:eye"}

  ```yaml
  ---
  title: "Article avec options d'affichage"
  # Image de couverture
  cover: /img/article-cover.jpg
  # Image de bannière
  banner: /img/article-banner.jpg
  # Couleur de fond personnalisée
  background_color: "#f8fafc"
  # Masquer la table des matières
  toc: false
  # Masquer la date
  hide_date: true
  # Masquer l'auteur
  hide_author: true
  ---
  ```
  ::
::

### Configuration des commentaires

::tabs{variant="card"}
  ::div{label="Commentaires" icon="lucide:message-circle"}

  ```yaml
  ---
  title: "Article avec commentaires"
  # Activer/désactiver les commentaires
  comments: true
  # Type de système de commentaires (si différent du défaut)
  comment_system: waline
  # Configuration spécifique aux commentaires
  comment_config:
    path: /custom/path/
    placeholder: "Laissez votre commentaire..."
  ---
  ```
  ::
::

### Configuration du partage

::tabs{variant="card"}
  ::div{label="Partage social" icon="lucide:share-2"}

  ```yaml
  ---
  title: "Article partageable"
  # Activer le partage social
  share: true
  # Plateformes de partage personnalisées
  share_platforms:
    - twitter
    - facebook
    - linkedin
    - telegram
  # Image pour le partage social
  og_image: /img/share-image.jpg
  ---
  ```

## Configuration des pages spéciales

### Page À propos

::tabs{variant="card"}
  ::div{label="about/index.md" icon="vscode-icons:file-type-markdown"}

  ```yaml
  ---
  title: À propos
  date: 2024-01-15
  type: about
  layout: about
  comments: false
  # Configuration spécifique à la page À propos
  about:
    avatar: /img/avatar.jpg
    name: "Votre nom"
    description: "Développeur passionné"
    location: "Paris, France"
    social:
      github: https://github.com/votre-nom
      twitter: https://twitter.com/votre-nom
      email: votre@email.com
  ---
  ```
  ::
::

### Page de tags

::tabs{variant="card"}

### Page de catégories

::tabs{variant="card"}

  ```yaml
  ---

  date: 2024-01-15
  ```

## Propriétés avancées

### Configuration SEO

::tabs{variant="card"}
  ::div{label="SEO avancé" icon="lucide:search"}

  ```yaml
  ---
  title: "Article optimisé SEO"
  # Méta-description personnalisée
  meta_description: "Description détaillée pour les moteurs de recherche"
  # Mots-clés spécifiques
  meta_keywords: [seo, optimisation, référencement]
  # Données structurées
  structured_data:
    type: Article
    author: "Votre nom"
    publisher: "Nom du site"
    dateModified: 2024-01-16
  # Canonical URL
  canonical: https://votre-site.com/article-original
  ---
  ```
  ::
::

### Configuration multilingue

::tabs{variant="card"}
  ::div{label="Multilingue" icon="lucide:languages"}

  ```yaml
  ---
  title: "Article en français"
  # Langue de l'article
  lang: fr
  # Versions alternatives
  alternate_versions:
    en: /en/article-title/
    es: /es/titulo-articulo/
  # Direction du texte
  text_direction: ltr
  ---
  ```

## Bonnes pratiques

### Conseils de rédaction

1. **Titres descriptifs** : Utilisez des titres clairs et descriptifs
2. **Descriptions concises** : Rédigez des descriptions de 150-160 caractères
3. **Tags pertinents** : Utilisez 3-7 tags pertinents par article
4. **Catégories logiques** : Organisez vos articles dans des catégories cohérentes
5. **Dates cohérentes** : Utilisez un format de date uniforme

### Validation du Front Matter

::alert{type="info"}
Utilisez un validateur YAML pour vérifier la syntaxe de votre Front Matter avant publication.
::

::alert{type="warning"}
Les erreurs de syntaxe YAML peuvent empêcher la génération correcte de votre site.
::
