## プラグインのインストール

DocSearch は algolia が提供する別の検索サービスです。詳細な申請と使用方法については [DocSearch ドキュメント](https://docsearch.algolia.com/) を参照してください

## テーマ設定ファイルの変更

```yml [_config.solitude.yml]
# 856行あたりと874行あたり
search:
  enable: true
  type: docsearch # local / algolia / docsearch
  tags:
    # - Solitude
    # - Hexo

···

  # DocSearch
  # https://docsearch.algolia.com/
  docsearch:
    appId:
    apiKey:
    indexName:
    option:
```
